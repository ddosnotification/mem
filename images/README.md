### Image Optimization

Before adding images:
1. Compress images using TinyPNG (https://tinypng.com)
2. Use WebP format for better performance
3. Provide fallback PNG for older browsers
4. Recommended dimensions:
   - Logo: 200x80px
   - Product cards: 400x400px
   - Hero images: 1920x1080px
   - Icons: 64x64px

### Branding Colors

When sourcing/editing images, maintain these brand colors:
- Primary Red: #ff0080
- Primary Pink: #ff6b9d
- Dark Red: #d4006d
- Light Pink: #ffb3d9
- Accent Purple: #8b5cf6
