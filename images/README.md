# Images Folder

This folder contains all visual assets for the Podperos website.

## Logo Files

- **logo.svg** - Current SVG logo (scalable, looks great on all devices)
- **logo.png** - Add your custom PNG logo here if you prefer PNG format

To use a PNG logo instead of SVG:
1. Replace `images/logo.svg` with `images/logo.png` in `index.html` (line ~46)
2. Recommended PNG size: 400x160px at 2x resolution (200x80px display size)
3. Transparent background recommended

## Recommended Additional Images

For enhanced visual experience, you can add:

- `hero-background.jpg` - Hero section background image
- `pouch-*.png` - Individual product images for nicotine pouches
- `vape-*.png` - Vape product images
- `vending-machine.png` - Photo of your vending machines
- `lips-visual.png` - Custom lips graphic (currently using SVG)
